jQuery Banner Rotator

API

Options

The banner rotator provides many customizable options. The following are all options grouped by categories.
(See setting options for directions on usage.)

Size

name type default description
responsive boolean true Enable responsive mode.
width number 1000 Width of the banner rotator.
height number 400 Height of the banner rotator.
thumbWidth number 32 Width of the thumbnails.
thumbHeight number 32 Height of the thumbnails.
thumbMargin number 3 Margin of the thumbnails.
buttonWidth number 32 Width of the buttons.
buttonHeight number 32 Height of the buttons.
buttonMargin number 3 Margin of the buttons.
sideButtonMargin number 0 Margin of the outside navigation buttons.
tooltipWidth number | string 'auto' Width of the tooltip. Use 'auto' for tooltip width to match content.
tooltipHeight number | string 'auto' Height of the tooltip. Use 'auto' for tooltip height to match content.
navThumbWidth number 100 Width of the navigation thumbs.
navThumbHeight number 75 Height of the navigation thumbs.

Slide Show

name type default description
autoPlay boolean true Start slide show automatically.
delay number 6000 The slide show's delay, in milliseconds.
startIndex number | string 0 The index of the slide to start on. (index uses zero-based numbering). Set 'random' for a random index.
pauseOnHover boolean false Pause slide show on hover and resume on hover out.
pauseOnInteraction boolean false Pause slide show on click of any control elements.
playOnce boolean false Play slide show once, stopping at last slide.
timer string 'pie' The timer indicator type - 'none' | 'bar' | 'pie'
timerPosition string 'right top' Where the timer is positioned - 'left top' | 'right top' | 'left bottom' | 'right bottom'

Transition

name type default description
cssTransition boolean true Enable css transition if supported by browser.
effect string 'fade' The image transition effect. See image effects for available effects.
duration number 800 The duration of the effect, in milliseconds.
easing string 'ease' The easing of the effect. Refer to site for more information about easing.
effectOnStart boolean true Enable initial starting effect.
columns number 1 The number of columns the effect consists of.
rows number 1 The number of rows the effect consists of.
interval number 100 The start delay between the pieces in the effect, in milliseconds.
Used only when effect is divided into multiple rows or columns.
direction string 'right' The effect's direction - 'up' | 'down' | 'left' | 'right'
Used only for directional type effects.
order string 'right' The starting order of the pieces in the effect. Used only when effect is divided into multiple rows or columns. See transition orders for available orders.
alternate boolean false If pieces in the effect alternate in effect directions. Used only for directional type effects divided into multiple rows or columns.
autoReverse boolean true When navigating to the previous slide, if the effect will be in reverse direction and order.
depth number | string 0 The z offset used during 3D transition effects. Use 'auto' to automatically calculate offset based on shape size.
shapeShading boolean true Enable shading for 3D transition effects.
shapeColor string '#333333' The background color of transition shapes, in hexadecimal notation.
shapeDepth number 0 The depth (thickness) of transition shapes. Used only when effect is set to 'flip'.

Ken Burns

name type default description
kbEffect string 'none' The Ken Burns effect of the images. See Ken Burns for available effects.
kbDuration number | string 'auto' The duration of the Ken Burns effect, in milliseconds. Use 'auto' for duration to equal to slide show's delay.
kbEasing string 'linear' The easing of the Ken Burns effect. Refer to site for more information about easing.

Control Elements

name type default description
cpanelPosition string 'left bottom' Where the control panel is positioned. See positions for available values.
cpanelOrientation string 'horizontal' The control panel's orientation - 'horizontal' | 'vertical'
cpanelOutside boolean false If the control panel is located outside of banner rotator's screen.
cpanelOnHover boolean false Display control panel on hover and hide on hover out.
thumbnails string 'number' The type of thumbnails - 'none' | 'bullet' | 'number' | 'text' | 'image'
selectOnHover boolean false Select slides on hover of thumbnails.
tooltip string 'text' The type of tooltip - 'none' | 'text' | 'image'
tooltipDelay number 0 The delay for tooltip to appear, in milliseconds.
navButtons string 'small' The type of navigation buttons - 'none' | 'small' | 'large' | 'outside'
navButtonsOnHover boolean false Show navigation buttons on hover and hide on hover out. Used only when navButtons is set to 'large' or 'outside'.
navThumbs boolean false Display the side navigation thumbs. Used only when navButtons is set to 'large' or 'outside'.
playButton boolean true Display the play button.
groupButtons boolean false Group together the thumbnails and buttons of the control panel.
cpanelGap number 0 The spacing between the screen and the outside control panel.
hideControl boolean false Hide control elements during image transition.

Layers

name type default description
layerEffect string 'fade' The layer's transition effect. See layer effects for available effects.
layerDuration number 500 The duration of the layer's transition effect, in milliseconds.
layerEasing string 'ease' The easing of the layer's transition effect. Refer to site for more information about easing.
layerDelay number 0 The start delay for the layer's transition effect, in milliseconds.
layerEffectOut string 'fade' The layer's transition out effect. See layer effects for available effects.
layerDurationOut number 500 The duration of the layer's transition out effect, in milliseconds.
layerEasingOut string 'ease' The easing of the layer's transition out effect. Refer to site for more information about easing.
layerDelayOut number 0 The start delay for the layer's transition out effect, in milliseconds.
layerSync boolean true Defer layer transition effect until image transition is complete.
layerOutSync boolean true Defer image transition until layer transition out effect is complete.
layerOnHover boolean false Display layers on hover and hide on hover out.

Miscellaneous

name type default description
imagePosition string 'fill' The position of slide image - 'fill' | 'fit' | 'center' | 'stretch' | 'none'
preload boolean false Preload all images before startup.
shuffle boolean false Shuffle slide order.
keyboard boolean true Enable keyboard control when in focus. Use the left and right arrow to navigate and spacebar to play/pause slide show.
mousewheel boolean false Enable mousewheel scroll navigation.
swipe string 'horizontal' Touch swipe orientation - 'none' | 'horizontal' | 'vertical'
swipeDirection string 'normal' Touch swipe direction - 'normal' | 'reverse'

Internationalization

name type default description
playText string 'play' Title text for play button.
pauseText string 'pause' Title text for pause button.
prevText string 'previous' Title text for previous button.
nextText string 'next' Title text for next button.

Methods

Once initialized, the banner rotator provides several methods for use in your javascript code.

play

Play the banner rotator's slide show.

Code example:

Invoking the play method:

$('#myRotator').bannerRotator('play');

pause

Pause the banner rotator's slide show.

Code example:

Invoking the pause method:

$('#myRotator').bannerRotator('pause');

togglePlay

Toggle the play state of the banner rotator's slide show.

Code example:

Invoking the togglePlay method:

$('#myRotator').bannerRotator('togglePlay');

prev

Go to the previous slide.

Code example:

Invoking the prev method:

$('#myRotator').bannerRotator('prev');

next

Go to the next slide.

Code example:

Invoking the next method:

$('#myRotator').bannerRotator('next');

to

Go to the slide at the specified index. (index uses zero-based numbering).

Code example:

Invoking the to method:

//go to slide at index 3
$('#myRotator').bannerRotator('to', 3);

option

Get the value associated with the option name.

Code example:

Invoking the option method:

//get banner rotator's width
$('#myRotator').bannerRotator('option', 'width');

destroy

Remove the banner rotator's functionality completely. This will return the banner rotator's elements back to its pre-initialized state.

Code example:

Invoking the destroy method:

$('#myRotator').bannerRotator('destroy');

Events

The banner rotator emits several events where you can assign callback functions or bind event listeners to.

create

Triggered immediately after the banner rotator is created.

Code Examples:

Initialize the banner rotator with the onCreate callback specified:

$('#myRotator').bannerRotator({
    onCreate: function(extra) {
        console.log('Banner rotator created.');
    }
});

Bind an event listener to the rotatorCreate event:

$('#myRotator').on('rotatorCreate', function(event, extra) {
    console.log('Banner rotator created.');
});

play

Triggered when the slide show plays.

Code Examples:

Initialize the banner rotator with the onPlay callback specified:

$('#myRotator').bannerRotator({
    onPlay: function(extra) {
        console.log('Play on slide ' + extra.index);
    }
});

Bind an event listener to the rotatorPlay event:

$('#myRotator').on('rotatorPlay', function(event, extra) {
    console.log('Play on slide ' + extra.index);
});

pause

Triggered when the slide show pauses.

Code Examples:

Initialize the banner rotator with the onPause callback specified:

$('#myRotator').bannerRotator({
    onPause: function(extra) {
        console.log('Pause on slide ' + extra.index);
    }
});

Bind an event listener to the rotatorPause event:

$('#myRotator').on('rotatorPause', function(event, extra) {
    console.log('Pause on slide ' + extra.index);
});

prev

Triggered when going to the previous slide.

Code Examples:

Initialize the banner rotator with the onPrev callback specified:

$('#myRotator').bannerRotator({
    onPrev: function(extra) {
        console.log('Current slide at ' + extra.index);
    }
});

Bind an event listener to the rotatorPrev event:

$('#myRotator').on('rotatorPrev', function(event, extra) {
    console.log('Current slide at ' + extra.index);
});

next

Triggered when going to the next slide.

Code Examples:

Initialize the banner rotator with the onNext callback specified:

$('#myRotator').bannerRotator({
    onNext: function(extra) {
        console.log('Current slide at ' + extra.index);

Bind an event listener to the rotatorNext event:

$('#myRotator').on('rotatorNext', function(event, extra) {
    console.log('Current slide at ' + extra.index);
});

change

Triggered on slide change, before image transition.

Code Examples:

Initialize the banner rotator with the onChange callback specified:

    onChange: function(extra) {
        console.log('Slide change to ' + extra.index);

Bind an event listener to the rotatorChange event:

    console.log('Slide change to ' + extra.index);

Triggered after slide has completed its image transition.

Code Examples:

Initialize the banner rotator with the onChanged callback specified:

Triggered immediately after a slide image has been loaded.

Code Examples:

Initialize the banner rotator with the onLoad callback specified:
