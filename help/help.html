jQuery Banner Rotator

Setup

Install

To install the banner rotator, first add the banner rotator's style sheet inside the <head> tag: 

<link href="css/banner-rotator.css" rel="stylesheet">

Then, add the jQuery and banner rotator javascript files:

<script src="js/jquery.min.js"></script>
<script src="js/jquery.banner-rotator.min.js"></script>

Once you've included the necessary files, the next step is to add the html markup for the banner rotator.

Html Markup

The banner rotator's markup is very simple and consists of a <div> tag containing a list of images:

<div id="myRotator" class="banner-rotator">
    <ul class="slides">
        <li>
            <img src="images/photo1.jpg"/>
        </li>
        <li> 
            <img src="images/photo2.jpg"/>
        </li>
        <li> 
            <img src="images/photo3.jpg"/>
        </li>
    </ul>
</div>

First, the container <div> tag should have class="banner-rotator" and an id of your choice. For the example, we use myRotator as the id. Then, within the <div> tag, there should be an <ul> tag with class="slides". This <ul> tag holds all the <li> tag, with each <li> representing an individual slide. In addition, each <li> tag should contain an <img> tag.

Initialize

After the markup is created, the last step is to initialize the banner rotator using the .bannerRotator() function: 

<script>
$(window).load(function() {
    $('#myRotator').bannerRotator();
});
</script>

When using this function, it should reference the id of the markup we created earlier. Also, make sure the function is called inside $(window).load() so that the page is ready for initialization.

Setting Options

Additionally, you can customize the settings of the banner rotator by passing in options to the .bannerRotator() function. There are many options available, please refer to options for the full list.

Example:

Initialize the banner rotator with the width and height options specified:

<script>
$(window).load(function() {
    $('#myRotator').bannerRotator({
        width:1200,
        height:400
    });
});
</script>

If an option is not passed into the function, then a default value is used instead.

Your banner rotator's basic setup is now complete, follow the rest of this guide for more customization instructions for the banner rotator.

Style Class

The banner rotator has several classes for customizing its style. When using a style class, it should be set to the banner rotator's containing <div> tag. (See style class for all available classes.)

Example:

Using the shadow class.

<div id="myRotator" class="banner-rotator shadow">
    <ul class="slides">
        ...
    </ul>
</div>

Images

To add an image to the banner rotator, simply insert an <img> tag within the slide's <li> tag: 

<li>
    <img src="images/photo.jpg"/>
</li>

You can also use lazy loading for the image by using the data-src attribute instead:

<li>
    <img data-src="images/photo.jpg"/>
</li>

Lazy Loading can help to reduce the initial download time by delaying loading until needed.

Thumbnails

You can add and customize text and image thumbnails for the banner rotator.

Text Thumbs

To add text for the thumbnail, use the title attribute for the <img> tag.

Example:

<li>
    <img src="images/photo.jpg" title="my photo"/>
</li>

Note that the option thumbnail should be set to 'text' during initialization to enable text thumbnails. (See control elements options for more info).

Image Thumbs

By default, the thumbnail will use the main image as the thumbnail image. However, you can set a different thumbnail image by using the data-thumb attribute for the <img> tag.

Example:

<li>
    <img src="images/photo.jpg" data-thumb="images/thumb.jpg"/>
</li>

Note that the option thumbnail should be set to 'image' during initialization to enable image thumbnails. (See control elements options for more info).

Tooltip

You can add and customize text and image tooltips for the banner rotator.

Text Tooltip

By default, the tooltip will use the thumbnail's text also as its tooltip text. However, you can set a different tooltip text by inserting the text inside a tag with class tooltip.

Example:

<li>
    <img src="images/photo.jpg"/>
    <span class="tooltip">
        lorem ipsum
    </span>
</li>

Image Tooltip

By default, the tooltip will use the thumbnail's image also as its tooltip image. However, you can set a different tooltip image by using the data-tooltip attribute for the <img> tag.

Example:

<li>
    <img src="images/photo.jpg" data-tooltip="images/tooltip.jpg"/>
</li>

Note that the option tooltip should be set to 'image' during initialization to enable image tooltips. (See control elements options for more info).

Navigation Thumbs

The banner rotator can display a preview thumbnail when hovering over the side navigation buttons. By default, this thumbnail will use the main image as the thumbnail image. However, you can set a different image by using the data-nav-thumb attribute for the <img> tag.

Example:

<li>
    <img src="images/photo.jpg" data-nav-thumb="images/navthumb.jpg"/>
</li>

Note that the option navThumbs should be set to true during initialization to enable navigation thumbs. (See control elements options for more info).

Links

To add a link for a slide, simply wrap the <img> tag with an <a> tag.

Example:

<li>
    <a href="http://www.google.com" target="_blank"><img src="images/photo.jpg"/></a>
</li>

Image Transition

You can customize the transition effect of an individual slide by using the following data-* attributes for the slide's <li> tag:

  • data-effect - the transition effect. (see image effects for available effects).
  • data-duration - the duration of the transition effect, in milliseconds.
  • data-easing - the easing of the transition effect.
  • data-delay - the slide change delay, in milliseconds.

Examples:

<li data-effect="fade" data-duration="500" data-easing="ease" data-delay="5000">
    <img src="images/photo.jpg"/>
</li>

Alternatively, you can use the shorthand attribute data-transition to set all four values at once:

<li data-transition="fade 500 ease 5000">
    <img src="images/photo.jpg"/>
</li>

The values set for an individual slide will override any corresponding values set in initialization. Please see transition options for more info.

Advance Transition

A transition effect can be customized further by using the following data-* attributes for the slide's <li> tag:

  • data-direction - the direction of the effect. (used only for directional effects).
  • data-columns - the number of columns the effect consists of.
  • data-rows - the number of rows the effect consists of.
  • data-interval - the start delay between the pieces in the effect, in milliseconds. (used only for effects divided into multiple rows or columns).
  • data-order - the starting order of the pieces in the effect. (used only for effects divided into multiple rows or columns, see orders for available values).
  • data-alternate - if pieces in the effect alternate in effect directions. (used only for directional effects divided into multiple rows or columns).
  • data-depth - the z offset for 3D effects.
  • data-shape-shading - if shading is used for 3D effects.
  • data-shape-color - the background color of shapes.
  • data-shape-depth - the depth (thickness) of shapes. (used only for 'flip' effect).

Example:

<li data-effect="rotate" data-columns="6" data-direction="down" data-order="right" data-alternate="true">
    <img src="images/photo.jpg"/>
</li>

The values set for an individual slide will override any corresponding values set in initialization. Please see transition options for more info.

Ken Burns

An image can have a Ken Burns effect that starts during the slide show. To configure the Ken Burns effect, use the following data-* attributes for the slide's <li> tag:

  • data-kb-effect - the Ken Burns effect. (see Ken Burns for available effects).
  • data-kb-duration - the duration of the Ken Burns effect, in milliseconds. Set value to "auto" to match slide show's delay time.
  • data-kb-easing - the easing of the Ken Burns effect.

Examples:

<li data-kb-effect="zoomIn" data-kb-duration="auto" data-kb-easing="linear">
    <img src="images/photo.jpg"/>
</li>

Alternatively, you can use the shorthand attribute data-ken-burns to set all three values at once:

<li data-ken-burns="zoomIn auto linear">
    <img src="images/photo.jpg"/>
</li>

The values set for an individual slide will override any corresponding values set in initialization. Please see Ken Burns options for more info.

Layers

You can add text and html layers on top of each slide. To add a layer, insert a tag with its content inside the slide's <li> tag. You can use almost any tags such as <span>, <div>, <p>, <h1> to <h6>, etc.

Examples:

Adding a layer:

<li>
    <img src="images/photo.jpg"/>
    <h1>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Adding multiple layers:

<li>
    <img src="images/photo.jpg"/>
    <!-- Layer 1 -->
    <h1>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
    <!-- Layer 2 -->
    <div>
        Quisque ultrices, dui a pellentesque blandit, nisl velit posuere enim.
        Nulla commodo urna non velit venenatis id aliquet lorem tincidunt.
    </div>
    <!-- Layer 3 -->
    <p>
        Fusce fringilla cursus purus dignissim elementum.  
        Phasellus vehicula ligula eu justo tempus eget adipiscing justo porttitor.
    </p>
</li>

Layer Style

You can adjust the style of the layer by setting the properties for the style attribute.

Example:

<li>
    <img src="images/photo.jpg"/>
    <h1 style="top:10px; left:25px; width:300px; height:auto; padding:5px; background-color:#00f;">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Some useful style include:

  • top - the top position of the layer.
  • left - the left position of the layer.
  • bottom - the bottom position of the layer.
  • right - the right position of the layer.
  • width - the width of the layer. (Use auto for auto width.)
  • height - the height of the layer. (Use auto for auto height.)
  • padding - the padding of the layer.
  • border - the border of the layer.
  • font-size - the font size.
  • color - the text color.
  • background-color - the background color.

Layer Transition

Each layer can have a transition effect when coming into view. You can customize the transition effect by setting the following data-* attributes for the layer tag:

  • data-effect - the layer's transition effect. (see layer effects for available effects).
  • data-duration - the duration of the transition effect, in milliseconds.
  • data-easing - the easing of the transition effect.
  • data-delay - the transition effect's start delay, in milliseconds.

Examples:

<li>
    <img src="images/photo.jpg"/>
    <h1 data-effect="fade" data-duration="500" data-easing="ease" data-delay="100">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Alternatively, you can use the shorthand attribute data-transition to set all four values at once:

<li>
    <img src="images/photo.jpg"/>
    <h1 data-transition="fade 500 ease 100">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

The values set for an individual layer will override any corresponding values set in initialization. Please see layer options for more info.

Layer Transition Out

Each layer can have a transition out effect when hiding from view. You can customize the transition out effect by setting the following data-* attributes for the layer tag:

  • data-effect-out - the layer's transition out effect. (see layer effects for available effects)
  • data-duration-out - the duration of the transition out effect, in milliseconds.
  • data-easing-out - the easing of the transition out effect.
  • data-delay-out - the transition out effect's start delay, in milliseconds.

Examples:

<li>
    <img src="images/photo.jpg"/>
    <h1 data-effect-out="slideUp" data-duration-out="500" data-easing-out="linear" data-delay-out="1500">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Alternatively, you can use the shorthand attribute data-transition-out to set all four values at once:

<li>
    <img src="images/photo.jpg"/>
    <h1 data-transition-out="slideUp 500 linear 1500">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

The values set for an individual layer will override any corresponding values set in initialization. Please see layer options for more info.
