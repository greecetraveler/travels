Appendix

Image Effects

name description directional CSS3 3D
none No transition effect.
random Random transition effect.
cover New image covers over previous image.
expand Image expands in size.
fade Image fades in.
flip Image flips over.
move Image moves into view.
push New image pushes previous image out.
rotate Cube rotation effect.
slide Similar to push, but in natural slide direction.
zoom Previous image zooms in and fades out, showing new image.

* CSS3 effects are only supported by modern browsers. A fallback effect is applied if feature is not supported.
